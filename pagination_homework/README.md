# Django Pagination Homework

Standalone homework for paginating Moscow bus station data from a UTF-8 CSV file.

Source dataset: https://data.mos.ru/opendata/752

## Run

```bash
pip install -r requirements.txt
python3 manage.py runserver
```

## Page

Open:

```text
http://127.0.0.1:8000/bus_stations/?page=1
```
